# Snake Game

A classic Snake game implemented in Python using the curses library for a terminal-based experience.

## Features

- Classic snake gameplay in the terminal
- Score tracking
- Game over screen
- Simple arrow key controls
- Platform independent

## Requirements

- Python 3.x
- curses (built-in on Unix-like systems)
- windows-curses (on Windows)

## Installation

1. Clone the repository:

   ```
   git clone https://github.com/yourusername/snake-game.git
   cd snake-game
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## How to Play

- Use arrow keys to control the snake
- Eat the food (█) to grow and increase score
- Avoid hitting the walls or yourself
- Press 'q' to quit

## Running the Game

Run the main script:

```
python main.py
```

## Controls

- Arrow Keys: Move snake
- q: Quit during game
- r: Restart after game over
- Q: Quit after game over
